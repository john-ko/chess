# SUPER GHETTO CHESS API

## Dependencies
- [Stockfish chess engine](https://github.com/official-stockfish/Stockfish)
- node

#### Stockfish installation
- `git clone git@github.com:official-stockfish/Stockfish.git` or via https `https://github.com/official-stockfish/Stockfish.git`
- `cd Stockfish/src`
- `make build ARCH=x86-64` ( builds on x86-64bit machines )
- `./stockfish` run stock fish to see if installed correctly

#### usage
`node index.js`
